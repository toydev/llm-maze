# LLMによる2D空間認識とナビゲーション能力の研究

## 1. 背景と目的

趣味プロジェクト「DQAI」（ドラゴンクエストIをAIが攻略）では、ワールドマップでの地理的移動に課題がある。LLMが局所的な情報だけでなく、大局的なマップ情報を理解して長期的な行動計画を立てることができない。例えば、北が岩山で塞がれているのにそちらへ進もうとする、といった挙動が見られる。

本プロジェクトは、シンプルな迷路環境でLLMの空間認識能力とその限界を検証し、DQAIへフィードバックするためのテストベッドである。

## 2. 探求する問い

LLMにどう世界を提示すれば空間構造を理解できるか → `PromptStrategy`で切り替え

## 3. 対象とするLLM

- Gemini 2.5 Flash: ベースライン
- Ollamaローカルモデル: 比較対象

## 4. アプローチ

- `PromptStrategy`を切り替えながらLLMの挙動を比較・分析
- 評価は模範解答（BFS算出）との正解率で定量化
- 正解率50%超が最低ライン（期待前進距離 `2p-1` が正になる条件）

## 5. コマンド

| コマンド | 説明 |
|----------|------|
| `npm run execute <model> [maze] [strategy]` | LLM実行・結果保存 |
| `npm run summary [model] [maze] [strategy]` | 全体統計を表示 |
| `npm run detail [file]` | マス毎の詳細分析（処理時間など）|
